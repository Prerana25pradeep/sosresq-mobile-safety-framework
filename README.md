# SOSResQ

SOSResQ is an Android-based mobile safety framework that combines automated emergency response, forensic evidence preservation and real-time geolocation broadcasting.

The system introduces an Autonomous Safety Protocol (Dead-Man Switch) that automatically triggers emergency alerts when a user fails to check in within a predefined time window. It also incorporates a Secure Evidence Vault for preserving digital evidence while maintaining forensic integrity and chain of custody.
